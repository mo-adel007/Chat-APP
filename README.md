# Chat Application

## Overview

The Chat Application is a real-time messaging platform that allows users to communicate with friends and groups seamlessly. This project showcases modern web development practices, incorporating technologies like React, Chakra UI, Socket.IO, and token-based authentication for security.

## Features

- **Real-Time Messaging:** Enjoy instant messaging with friends and groups.
- **User-Friendly Interface:** Experience a smooth and intuitive communication platform.
- **Typing Indicators:** Dynamic chat experience with typing indicators.
- **Profile Management:** Personalize your profile details.
- **Create and Manage Group Chats:** Collaborate with friends in group chats.
- **Secure Login:** Token-based authentication ensures secure access to the application.

## Technologies Used

- **React:** This project leverages React for building dynamic and responsive user interfaces.
- **Chakra UI:** Chakra UI is used to create stunning and consistent designs.
- **Socket.IO:** Socket.IO powers real-time chat functionality for instant updates.
- **Axios:** Axios is used for efficient HTTP requests to the backend.

## Setup and Usage

1. Clone the repository to your local machine.

```
git clone https://github.com/your-username/chat-application.git
```

2. Install dependencies for the frontend.

```
cd chat-application
npm install
```

3. Start the development server.

```
npm start
```

4. Open the application in your web browser at `http://localhost:3000`.

## Folder Structure

- **`/src`**: Contains the application source code.
  - **`/components`**: React components used in the application.
  - **`/context`**: Context API for state management.
  - **`/config`**: Configuration files and utility functions.
  - **`/styles`**: Stylesheets and CSS files.
  - **`/assets`**: Images, icons, and other assets.
- **`/public`**: Public assets like HTML files and the application's favicon.
